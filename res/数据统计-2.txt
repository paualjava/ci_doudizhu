例如：建1小时房间，玩3局牌，每一局中有：
1.牌桌操作（发牌，翻牌，转牌，河牌，摊牌），
2.用户操作（加注（全押，1/2底池，底池），跟注，看牌，弃牌，下注），


牌型：同花大顺，同花顺，四条（金刚），葫芦，同花，顺子，三条，两对，一对，高牌
底池：每一个牌局里众人已押上的筹码总额，也即该局的奖金数目
边池：当某人全押的时候，一个边池就会形成。这个边池包含到这个时候为止众人所已经押上的筹码。全押的这个玩家若赢了这一局，只能赢走边池里的奖金，而不能应走在他全押以后其余人加上的注码。
入局：一局中任何时间第一次加入钱进去（大小盲是强制入局，拿到两张手牌直接弃牌不是入局，拿到两张手牌加了注后没经过翻牌又弃牌是入局，一局中多次加钱进去只算一次入局）


术语：
翻牌率，转牌率，河牌率，入池率，弃牌率，
入局率，加注率，加注弃牌率
单手最大(U币)，单局最大(U币)，总手数
激进度
最大牌型
最大底池


计算：

翻牌率 = 总翻牌数/总局数
河牌率 = 总河牌数/总局数
转牌率 = 总转牌数/总局数
摊牌率 = 总摊牌数/总局数
入池率 = 翻牌率 + 转牌率 + 河牌率
1 = 入池率 + 弃牌率


入局率 = 总入局数/总局数

加注率 = 总加注数/总局数
加注弃牌率 = 总加注并有弃牌数/总局数

胜率 = 总赢数/总局数

单手最大(U币)　= 用户一次操作（下注或加注）历史最大一次U币值
单局最大(U币)　= 用户一局内投入的总值（下注、加注总值）历史最大U币值
总手数　= 总局数

激进度 = (下注次数+加注次数)/跟注次数

最大牌型 = 历史上玩家拿到过的 且 这牌型经过摊牌 的最大的牌型

最大底池 = 历史上玩家经历过的最大底池

计算公式：

跟注值 = 当前最大值 - 当前行动者桌上已有下注值
最小加注值 = 当前最大差值 + 跟注值


当前桌上最大值 = MAX(行动者下注值 + 当前行动者桌上已有下注值 , 当前桌上最大值)
当前桌上最大差值 = MAX(行动者下注值 + 当前行动者桌上已有下注值- 当前桌上最大值 , 当前桌上最大差值)

看牌 = 跟注值 == 0

allin值 ：当前操作加完所有剩下的注
有效加注：allin值 < 最小加注值
